# Consultation Booking Application (Flutter)

## Overview
A mobile application developed to streamline the process of booking consultations between students and administrators. The system allows users to register, log in, and schedule appointments efficiently.

## Features
- User registration and login (Student & Admin)
- Appointment booking and management
- Profile management
- Role-based navigation
- User-friendly mobile interface

## Technologies Used
- Flutter (Dart)
- Firebase (Authentication & Database)
